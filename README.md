# NASA Landing Page

## Descripción

Frontend landing page inspired by NASA aesthetics, focused on layout, typography, visual hierarchy and responsive styling.

## Tecnologías

- HTML
- CSS

## Objetivo

Practicar una landing page visual con estética tecnológica y jerarquía clara.

## Funcionalidades

- Hero visual
- Composición temática
- Tipografía y espaciado
- Maquetación con HTML/CSS

## Aprendizajes

Mejora de composición visual, estructura semántica y estilos CSS.

## Mejoras futuras

Añadir versión responsive refinada, animaciones sutiles y demo online.
